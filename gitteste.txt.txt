versao 2
1- Para clonar um projeto do repositorio do github no repositorio local, primeiro é preciso criar (enviar) um 
projeto para a nuvem (repositorio no github).

2 - Depois que o projeto estiver na nuvem, para clonar ele, é preciso abrir o terminal e digitar o comando:
git clone <URL-do-repositorio>.

3 - Se o comando for executado com sucesso, vai ser criada altomaticamente uma pasta com o nome do seu repositorio
e nela estara os projetos que nele estão.

4- Então é preciso volta para o terminal e entrar tanto na pasta que foi últilizada para mandar os projetos para 
o repositorio do github e ao mesmo tempo na pasta clonada.

5- Quando estiver nas duas pasta será preciso fazer o mesmos comandos para enviar um projeto do repositorio local
para o repositorio do github, esses comandos devem ser executados depois que as modificações forem feitas.

6- Se não for retornado nenhum problema o arquivo clonado será enviado para o repositoriodo github.
